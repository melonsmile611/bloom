# 爆炸之年

**Date:** May 28, 2026
**Style:** 王小波风格散文

---

我一直觉得，人的一生大概是这样分配的：有些年份什么都不发生，另一些年份所有的事情挤在一起来轰炸你。去年是我的本命年，我压抑了整整一年，像一只被关在笼子里的动物，规规矩矩，战战兢兢。今年不知道哪里来的一股气，突然就炸开了。

先是换了手机。我把 Pixel 9 Pro 卖给了一个叫 Mobile Monster 的网站，拿了七百二十块钱。这件事让我意识到，任何东西都是有价格的，包括你曾经爱不释手的那块屏幕。然后我买了一个叫 Nothing Phone 的手机。"Nothing"这个名字起得很哲学，我喜欢有点小炫酷的东西，没办法，这是我的弱点。

同一天，我换了眼镜，度数加深了。同一天，网络升级成了 NBN 100。同一天，我妈帮我定的防摔倒手表也到了——那种戴在手腕上、随时可以呼救的手表。我站在那一堆新东西中间，忽然有点哑然失笑：我到底是在生活，还是在给自己不断升级配置？

我妈的身份也在这段时间下来了。这件事比手表重要得多，但我说它的语气好像和说"换了网络"差不多。人就是这样，真正重要的事反而说不出什么，只是觉得：哦，好了。

---

最近迷上了用 Claude Code 写程序。它是一个住在终端里的 AI，你跟它说话，它就帮你写代码。我发现把它配合 VSCode 一起用，对话都能被记录下来，比在黑乎乎的终端里聊天强多了——内容不会一晃而过，你可以看见自己说过什么、它做了什么，这种可见性让我安心。

我还有另一个 AI，叫 Hermes，负责帮我传话、做事。有一天我让他去调用 Claude Code 来写程序。他说他可以。我等着。结果还是不行。我其实一点也不意外——能做到才奇怪——但不知道为什么，我当时挺开心的，就是那种"哦你也不行啊"的释然感。

我让 Hermes 写好 prompt，写好分镜脚本，然后拿去让图像生成工具 Antigravity 自己生成图片，再生成动画。会搞成什么样？我不知道。但这本身就很有意思——你布置了一个任务，然后等着看它会爆炸成什么形状。

---

我用 Claude Code 做了一个治愈系小游戏的网页。游戏里用了我喜欢的那种风格的图片，渲染得很漂亮。游戏本身嘛——有点小无聊。但这正是我追求的：安静，没有刺激，像在发呆。

有个树叶游戏有个 bug，树叶不会落下来。本来想让 Claude Code 修，后来想想，算了。一片停在空中不肯落下的叶子，说不定也有它的道理。

---

我还在搞开店的事。想做定制帆布帽，也想做帽衫，要和国内的厂家沟通。复杂得很，玲玲在帮我处理，我觉得她快被搞筋疲力尽了。开店不是一件容易的事，我早该知道的，但人总是要试了才信。

现在我也不知道何去何从。能做的都做了，能试的都试了。接下来顺其自然吧。"顺其自然"这四个字，我以前觉得是认输，现在觉得是另一种有趣的活法——你往前走，但不抓着结果不放。

---

我去年本命年压抑了一整年，今年全炸出来了：换手机、换眼镜、换网络、开店、妈妈身份到手、手表到手、做游戏、学新工具。这些事情挤在一起，看起来很热闹，但热闹的底下其实是一种平静——不是那种压抑的平静，而是找回了感觉的那种。

就这样。今天过得不错。
