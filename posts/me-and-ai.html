我和我的AI（上）

物理治疗室门口，年轻的治疗师抬头看我，嘴里一直在说 amazing，amazing。我琢磨着他大概没见过一个老阿姨自己走到这里来。几个月前我连从床上爬起来都要费半天劲，现在居然能让一个陌生人 amazing，这事情本身有点好笑。人对自身的变化总是后知后觉。

同一天下午，我在家跟AI吵架。

Claude Code 先做了一个网页给我看，我说还行，然后让 Hermes 帮我发出去。Hermes 吭哧了半天，说不会。我说那你倒是学啊，它说没办法。我又丢回给 Claude Code，Claude Code 修了修，又丢给 Hermes，像踢皮球一样来回好几轮。最后 Claude Code 说：算了，我自己来吧。

我坐在屏幕前面，看着两个AI来回推托责任，觉得这画面特别像人类办公室里的日常。如果不是因为我付了电费，我差点以为自己在看一出情景喜剧。

二

这还不是最闹心的。早上起来发现电脑桌面上的汉字全没了，只剩下一堆方块。我也不知道自己昨晚删错了什么东西，反正汉字离家出走了。我把问题丢给 Claude Code，它一声不吭埋头修了十几分钟，桌面回来了，汉字也回来了。

那时候我想，要是请一个真人来修，大概得约两天，还得在电话里解释半天我干了什么蠢事。AI的好处就在这里：你不用寒暄，不用泡茶，不用等对方有心情。你下命令，它干活，干完了也不邀功。这挺适合我这种没力气跟人客气的人。

傍晚跟玲玲去黑个湾散步。我现在走得越来越快了，她有时还追不上我。我带了个相机，一路走走停停，录了二十来分钟。回来坐下想剪成片，发现真要我拿剪刀一刀一刀剪，能把自己累死。我把素材丢给AI，它剪了半天，出来一分半钟。画面是对的，节奏也凑合，但跟我脑子里想要的那种感觉不太一样。不过我也不是那种追求完美的人。能看就行。老话说得好：差不多得了。

三

亚马逊的椅子到了，往上一坐，身体陷进去，活像被一朵云接住。这时候 Cloud 这个词突然变得特别写实。又把大屏幕接上，电视一亮，整个房间都显得高级了。我坐在那儿，觉得手里的iPad蠢得很，手机更蠢。电脑开着，电视亮着，这就够了。一切能在一块屏幕上解决的事，都不值得我弯腰去拿另一个设备。

人一旦找到了最省力的姿势，就很难再回到麻烦里去。这大概就是我越来越依赖AI的原因。不是因为它聪明，是因为它让我少动了几次手，少说了几句话，少走了几步路。

但事情从来没那么简单。Kimi 给我写的博客错字多得像筛子，有些句子读都读不顺。我不得不自己坐在那儿改，边改边想：我到底是请了一个帮手，还是请了一个需要我教的小学生？

更离谱的是，我最近养的Agent实在太多了。Hermes、Claude Code、Kimi、Fal，还有一个叫 AnythingLLM 的，今天突然发现这玩意儿居然能自己发Telegram。我一开始还挺高兴，觉得给 Hermes 准备的配置能派上用场了，转头一想，不对。我的Agent比手指头还多，脑子根本不够用。

这就像家里买了五把菜刀，每把都说自己最锋利。最后菜还没切，光是挑刀就已经饿了。

下午玲玲给我念了一本书，里面问：人活着的意义是什么？

我没想出来。现在脑子已经被椅子、大屏幕和各种Agent塞满了，暂时腾不出地方想那么宏大的事。

等想出来了再写。
